## 功能模块清单

### 1. 账户管理 (Account)

| 功能 | 接口 | 状态 |
|------|------|------|
| 获取账户列表 | GET /v1/accounts | ✅ |
| 获取账户详情 | GET /v1/accounts/{id} | ✅ |
| 创建账户 | POST /v1/accounts | ✅ |
| 更新账户 | PUT /v1/accounts/{id} | ✅ |
| 删除账户 | DELETE /v1/accounts/{id} | ✅ |
| 获取账户交易流水 | GET /v1/accounts/{id}/transactions | ✅ |
| 调整账户余额 | POST /v1/accounts/{id}/adjust-balance | ✅ |

**实体类**: Account
- 账户ID、用户ID、账户名称、账户类型
- 图标、颜色、初始余额、信用额度
- 计入净资产、允许转账、余额预警

---

### 2. 交易管理 (Transaction)

| 功能 | 接口 | 状态 |
|------|------|------|
| 获取交易列表 | GET /v1/transactions | ✅ |
| 获取最近交易 | GET /v1/transactions/recent | ✅ |
| 获取交易详情 | GET /v1/transactions/{id} | ✅ |
| 创建交易 | POST /v1/transactions | ✅ |
| 更新交易 | PUT /v1/transactions/{id} | ✅ |
| 删除交易 | DELETE /v1/transactions/{id} | ✅ |
| 退款处理 | POST /v1/transactions/{id}/refund | ✅ |
| 重复交易检测 | POST /v1/transactions/check-duplicate | ✅ |
| 搜索筛选交易 | GET /v1/transactions/search | ✅ |

**核心功能**:
- 复式记账（借贷平衡校验）
- 简化模式（自动生成借贷分录）
- 完整模式（手动录入分录）
- 退款处理（反向分录）
- 交易类型：支出、收入、转账、还款

**实体类**: Transaction, Entry

## 技术特性

### 1. 复式记账系统
- 借贷平衡校验
- 自动分录生成（简化模式）
- 手动分录录入（完整模式）
- 退款反向分录

### 2. 数据安全
- JWT认证
- 用户数据隔离
- 软删除机制

### 3. 分页与查询
- 统一分页响应格式
- 多条件组合搜索
- 排序支持

### 4. 异常处理
- 统一错误码
- 全局异常处理器
- 业务异常封装
